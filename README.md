# tor

## Run

Run "Tor" shortcut and don't close opened terminal window.

"Bootstrapped 100% (done): Done" that means it is connected successfully to the tor network.

## Use

Set "Socks5" proxy of your software to use this tunnel.
Server IP:  127.0.0.1
Server Port: 9050
